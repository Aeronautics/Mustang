Mustang Disposition Library

Hello! Welcome to the Mustang Disposition Library. This library is composed of several reusable patterns for common HTML5 tags.

For the nav element

disposition: mustang-nav-bar;

Navigation bars are essential to the Mustang workflow. They're nice and big, good for clicking and touching, can handle form components and spans over multiple lines if necessary.
