Aeronautics\Mustang

A reusable Stylus CSS Framework

Aeronautics\Mustang is a not yet fully featured, all-optional, modular CSS framework written in Stylus. This page is styled by this:

html
  area 100%

body
  area 55.5em 100%
  position relative top center
  font 14px Arial, Helvetica, FreeSans, sans-serif
  background #EEE
  color #555

header
  area 100% 5.5em

header, article h2
  border-bottom 1px solid #CCC

article
  width 12col 15col

nav
  margin 5gutter
  width 3col 15col

ul
  padding-left 1em

li
  padding-top 0.5em

section
  margin 0.35gutter
  width 2col 4col

header, h2, a
  color darken(teal, 25%)

pre
  padding 0.5em
  border 1px solid #CCC

Reference

area

Simply combines width and height.

body
  area 60em
    //width:  60em
    //height: 60em
  area 60em 70em
    //width:  60em
    //height: 70em

position absolute

Beautifies the native position property.


position absolute top left
  //position: absolute
  //top: 0
  //left: 0
position absolute bottom right
  //position: absolute
  //bottom: 0
  //right: 0

position absolute top left 1em 2em
  //position: absolute
  //bottom: 1em
  //right: 2em

position relative

Beautifies the native position property.


position relative center
  //position: relative
  //margin: 0 auto
position
 relative left
  //position: relative
  //margin: 0 auto 0 0

width

Makes the native with shine with the new kind of units col and gutter for CSS grids.

half = 50%
section
  width 10% half
    //10% of 50%
    //width: 5%

section
  width 10% 50%
    //10% of 50%
    //width: 5%

section
  width 6col 12col
    //Grid system model
    //width: 50%
    //float: left
    //overflow: hidden
    //display: inline

section
  margin 40gutter
  width 6col 12col
    //Grid system model
    //width: 49% //around that
    //margin: 0.5% //around that
    //float: left
    //overflow: hidden
    //display: inline
