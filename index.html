Aeronautics\Mustang

A reusable Stylus CSS Framework

Aeronautics\Mustang is a not yet full featured, all-optional, modular CSS framework written in Stylus. This page is styled by this:

header-size = 14%
nav-size = 18%
html
  area 100%
body
  position relative
  area 50em 100%
  margin 0 auto
  font 14px Helvetica, Arial, FreeSans, sans-serif
  background #EEE

header
  text-align center
  area 100% header-size
  letter-spacing -0.08em
  border-bottom 0.02em solid #AAA

nav
  area left nav-size top header-size 100%

article
  area left nav-size 100% top header-size 100%

header, nav, article, footer
  position absolute

header, a, body
  color #444

pre
  padding 1em
  background #DDD

See the generated CSS file. Mustang is about 20% shorter and much more easier to maintain, all those percentages are managed by us.

Currently, only area is implemented for managing absolutely positioned elements. We're currently working on our roadmap:

  • Let area management deal with mixed absolute and relative positions
  • Granular and global reset/normalize mixins
  • Parametrized color and font themes
  • Semantic grid based on proportions inspired by semantic.gs

Area usage is:

  • area left|right x1 x2 top|bottom y1 y2
  • area x1 x2 y1 y2
  • Consecutive x's and y's can be contracted like area left 70%, which is equivalent to area left 0 70%
