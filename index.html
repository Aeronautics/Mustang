Aeronautics/Mustang

A semantic-friendly CSS extension. It doesn't enforce any specific HTML markup. You're free.

Let's start by eating our own dogfood. This page is styled only by these rules:

@import 'mustang.styl'
@import 'prefixes/all.styl'
@import 'uses/normal.styl'

{
  use: normal, fonts spacing, html h1 h2 p ul li;
  use: normal, html5-compat, sectioning;
}
body {
  area: 65.5em 100%;
  position: relative top center;
}
article,
nav,
section {
  position: grid;
}
article {
  width: 8col;
  float: right;
}
nav {
  width: 4col;
}
ul {
  padding: 0 1em;
  margin: 0;
}
li {
  margin: 0.4em 0;
}
section {
  width: 1col 2grid 10gutter;
}
section code {
  color: darkblue;
}
section ins {
  text-decoration: none;
  color: #777;
}

Reference

use

Aplies groups of styles according to a pattern, acts over children.


use: normal, fonts, h1;
  //generates:
  //h1 {
  //  font-family: sans-serif;
  //  font-size: 100%;
  //  text-adjust: 100%; 
  //}

Useable modules can be created, normal is a module for granular CSS normalization.

use: normal, [features], [children]

Features
  • fonts
    • font-family
    • font-size
    • text-size-adjust
  • spacing
    • margin

Aplies CSS normalization on all elements listed on [children].

area

Simply combines width and height.


area: 60em;
  //generates:
  //width:  60em
  //height: 60em
area: 60em 70em;
  //generates:
  //width:  60em
  //height: 70em

position absolute

Beautifies the native position property.


position: absolute top left;
  //generates:
  //position: absolute
  //top: 0
  //left: 0
position: absolute bottom right;
  //generates:
  //position: absolute
  //bottom: 0
  //right: 0
position: absolute top left 1em 2em;
  //generates:
  //position: absolute
  //bottom: 1em
  //right: 2em

position relative

Beautifies the native position property.


position: relative center;
  //generates:
  //position: relative
  //margin: 0 auto
position:
 relative left;
  //generates:
  //position: relative
  //margin: 0 auto 0 0

width and position: grid

Makes the native with shine with the new kind of units col and gutter for CSS grids.


section {
  width: 6col 12grid;
    //generates
    //6 out of 12 columns
    //width: 50%
  position: grid;
    //float: left
    //overflow: hidden
    //display: inline
}

section {
  width: 6col 12grid 40gutter;
    //Grid system model
    //6 out of 12 columns with gutter
    //width: 49% //around that
    //margin: 0.5% //around that
  position: grid;
    //float: left
    //overflow: hidden
    //display: inline
}
